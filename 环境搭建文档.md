# MIS系统开发环境搭建文档

## 1. 环境搭建步骤

### 1.1 前端环境搭建

1. **Node.js环境配置**
   - 检查Node.js版本：`node -v`（已安装v25.9.0）
   - 安装pnpm：`npm install -g pnpm`

2. **SoybeanAdmin前端框架安装**
   - 进入前端目录：`cd frontend`
   - 克隆SoybeanAdmin仓库：`git clone https://github.com/soybeanjs/soybean-admin.git .`
   - 安装依赖：`pnpm install`

3. **前端配置**
   - 修改`.env`文件，配置API基础URL：
     ```
     VITE_SERVICE_BASE_URL=http://localhost:8080
     ```
   - 修改`vite.config.ts`文件，设置端口为8080：
     ```typescript
     server: {
       host: '0.0.0.0',
       port: 8080,
       open: true,
       proxy: createViteProxy(viteEnv, enableProxy)
     },
     ```

### 1.2 后端环境搭建

1. **PHP环境配置**
   - 检查PHP版本：`php -v`（已安装8.5.4）
   - 检查Composer版本：`composer -v`（已安装2.9.5）

2. **CodeIgniter框架部署**
   - 进入后端目录：`cd backend`
   - 安装依赖：`composer install`

3. **数据库连接设置**
   - 修改`app/Config/Database.php`文件：
     - 设置默认数据库连接组为`btdc`：
       ```php
       public string $defaultGroup = 'btdc';
       ```
     - btdc连接配置已存在，使用以下参数：
       ```php
       public $btdc = [
           'DSN'      => '',
           'hostname' => '114.113.154.36',
           'username' => 'data',
           'password' => 'MYSQL.data2023',
           'database' => 'data',
           'DBDriver' => 'MySQLi',
           'port'     => 45608,
       ];
       ```

4. **CORS配置**
   - 修改`app/Config/Cors.php`文件，允许前端访问：
     ```php
     'allowedOrigins' => ['http://localhost:8080'],
     'supportsCredentials' => true,
     'allowedHeaders' => ['Content-Type', 'Authorization', 'X-Requested-With'],
     'allowedMethods' => ['GET', 'POST', 'PUT', 'DELETE', 'OPTIONS'],
     ```

5. **API接口配置**
   - 修改`app/Config/Routes.php`文件，添加API路由：
     ```php
     // API routes
     $routes->group('api', function($routes) {
         $routes->get('test', 'Api::test');
         $routes->post('login', 'Api::login');
     });
     ```
   - 创建`app/Controllers/Api.php`控制器，实现测试接口

## 2. 验证方法

### 2.1 后端服务验证

1. **启动后端服务**
   - 进入后端目录：`cd backend`
   - 启动内置服务器：`php spark serve`
   - 服务默认运行在 `http://localhost:8080`

2. **测试API接口**
   - 访问测试接口：`http://localhost:8080/api/test`
   - 预期返回：
     ```json
     {
       "code": "0000",
       "msg": "Success",
       "data": {
         "message": "API test successful",
         "timestamp": 1234567890
       }
     }
     ```

### 2.2 前端服务验证

1. **启动前端服务**
   - 进入前端目录：`cd frontend`
   - 启动开发服务器：`pnpm dev`
   - 服务默认运行在 `http://localhost:8080`

2. **验证前端页面**
   - 打开浏览器访问：`http://localhost:8080`
   - 预期看到SoybeanAdmin登录页面

3. **验证前后端通信**
   - 在前端页面中测试登录功能
   - 检查浏览器控制台，确认API请求成功

## 3. 常见问题排查

### 3.1 后端问题

1. **数据库连接失败**
   - 检查`Database.php`中的连接参数是否正确
   - 确认MySQL服务是否正常运行

2. **API接口404**
   - 检查路由配置是否正确
   - 确认控制器文件是否存在且命名正确

### 3.2 前端问题

1. **依赖安装失败**
   - 确保使用pnpm安装依赖
   - 检查网络连接是否正常

2. **API请求失败**
   - 检查CORS配置是否正确
   - 确认后端服务是否正在运行
   - 检查前端API基础URL配置是否正确

### 3.3 前后端通信问题

1. **跨域错误**
   - 检查后端CORS配置
   - 确认前端请求的URL格式正确

2. **API响应格式错误**
   - 确保后端返回的JSON格式符合前端预期
   - 检查前端`isBackendSuccess`函数的逻辑

## 4. 项目结构

```
mis/
├── backend/          # 后端CodeIgniter项目
│   ├── app/          # 应用代码
│   ├── public/       # 公共文件
│   ├── vendor/       # 依赖包
│   └── writable/     # 可写目录
├── frontend/         # 前端SoybeanAdmin项目
│   ├── src/          # 源码
│   ├── build/        # 构建配置
│   ├── packages/     # 内部包
│   └── public/       # 公共文件
└── 环境搭建文档.md    # 环境搭建文档
```

## 5. 开发流程

1. **启动后端服务**：`cd backend && php spark serve`
2. **启动前端服务**：`cd frontend && pnpm dev`
3. **开发API接口**：在`backend/app/Controllers`中创建控制器
4. **开发前端页面**：在`frontend/src/views`中创建页面
5. **测试API接口**：使用Postman或浏览器测试
6. **构建前端项目**：`cd frontend && pnpm build`

## 6. 技术栈

- **前端**：Vue 3 + TypeScript + Vite + Naive UI + SoybeanAdmin
- **后端**：PHP 8.5.4 + CodeIgniter 4.7.2
- **数据库**：MySQL
- **工具**：Composer, pnpm, Git
